예제로 사용된 url 주소 :

http://codingeverbody.com/codingeverbody_html_tutorial/url_72/example2.html?move=view>#root

부분 명칭 설명
http:// scheme 통신에 사용되는 방식,프로토콜
codingeverbody.com hosts 자원이 위치하고 있는 웹 서버의 이름, 도메인이나 ip가 사용됨
/codingeverbody_html_tutorial/url_72/example2.html url-path 루트 디렉토리부터 자원이 위치한 장소까지의 티렉토리 파일명
?move=view query 웹서버에 넘기는 추가적인 질문
#root bookmark 하이퍼링크를 클릭했을 때 특정 위치로 이동하기 위해서 사용
